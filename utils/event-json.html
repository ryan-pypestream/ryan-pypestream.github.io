Event Name
Use noun + past participle, in Title Case
Examples: "Product Purchased", "Refund Requested"
If event name is not provided, the name of the node will be used for the event name.

Properties
Use snake_case with no spaces or uppercase letters, like: "product_name", "refund_amount".
The display names can be changed in Analytics to show "Product Name" or "Refund Amount"
You can insert variables in property values, like this: "User Name is {USER_NAME}".
Global Props Set
Global properties are applied to all events in all future nodes once a node with "global_props_set" is visited.
Global Props Unset
Once a node with "global_props_unset" is encountered, the specified global properties are removed and will no longer be applied to future events.
Retention Days
Retention days specify for how many days an utterance-property retains its value in database. After retention period is passed, the value will be redacted to "<--expired-->"
